# Deployment scripts

| Production path | Script |
|-----------------|--------|
| Local VM + Cloudflare | `./scripts/deploy-local-vm.sh` |
| AWS / any cloud VM | `./scripts/deploy-aws.sh` |

```bash
./scripts/deploy-local-vm.sh all
./scripts/deploy-aws.sh all
```

Guide: [docs.authengine.org/deployment](https://docs.authengine.org/deployment/)
